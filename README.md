# Quiz Game in C

A simple command-line Quiz Game written in C.

## Features
- Multiple questions with options
- Score tracking
- Instant feedback

## Technologies Used
- C Programming

## How to Run
1. Compile:
   gcc main.c -o quiz
2. Run:
   ./quiz

## Example
Q: What is 2 + 2?  
A. 3  
B. 4  
C. 5  
D. 6  
Answer: B
